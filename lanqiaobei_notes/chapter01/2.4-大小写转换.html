2.4.1 C++ islower/isupper函数详解

1.函数核心概述

islower 和 isupper 是 C++ 标准库提供的字符分类专用函数，核心功能如下：

  • islower(char c)：检查传入的字符 c 是否为小写英文字母（a-z）
  • isupper(char c)：检查传入的字符 c 是否为大写英文字母（A-Z）

2.使用前置条件

  1. 头文件依赖：必须包含 <cctype> 头文件（该头文件封装了所有C++字符分类与转换函数），也可使用 #include <bits/stdc++.h> 万能头文件间接包含。
  2. 参数类型：通常接收 char 类型字符作为参数（底层兼容 int 类型，传入 ASCII 码值也可生效）。
  3. 返回值类型：返回 bool 类型（布尔值）：
    • 满足条件（小写/大写字母）：返回 true（逻辑真）
    • 不满足条件（非小写/非大写字母，如数字、符号、空格等）：返回 false（逻辑假）

5.关键说明

  1. 判断范围限制：两个函数仅对英文字母生效，对中文、日文等非英文字符，一律返回 false。
  2. 空字符/特殊字符：传入 '\0'（空字符）、空格、制表符等，返回 false。
  3. 实用场景：常用于字符串处理（如密码强度校验、文本格式规范化、字母大小写统计等）。

2.4.2 tolower/toupper函数详解

1.函数核心功能

tolower 和 toupper 是 C++ 标准库的字符转换函数，作用是将字母在大小写之间转换：

  • tolower(char ch)：若 ch 是大写字母（A-Z），则转换为对应的小写字母；否则保持原字符不变。
  • toupper(char ch)：若 ch 是小写字母（a-z），则转换为对应的大写字母；否则保持原字符不变。

2.使用说明

  1. 头文件依赖：需包含 <cctype> 头文件（或万能头 <bits/stdc++.h>）。
  2. 参数与返回值：参数为 char 类型字符，返回转换后的 char 类型字符。
  3. 非字母处理：若传入的不是字母（如数字、符号），函数不会对其进行修改，直接返回原字符。
