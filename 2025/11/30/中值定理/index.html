三大微分中值定理

Created2025-11-30|Updated2025-11-30
|Post Views:

三大中值定理总结（含考研真题示例，每题1道例题）

一、罗尔定理（Rolle’s Theorem）

1. 核心定义

若函数 $f(x)$ 满足 3个条件：

  • 闭区间连续：$f(x) \in C[a,b]$（在$[a,b]$上连续）；
  • 开区间可导：$f(x) \in D(a,b)$（在$(a,b)$内可导）；
  • 端点值相等：$f(a) = f(b)$；
    则存在 $\xi \in (a,b)$，使得 $f’(\xi) = 0$。

几何意义：连续可导曲线在区间端点函数值相等时，曲线上至少有一点的切线与x轴平行（水平切线）。

2. 考研真题示例（2017年数学一/二/三）

设函数 $f(x)$ 在$[0,1]$上连续，在$(0,1)$内可导，且 $f(1) = 0$。证明：存在 $\xi \in (0,1)$，使得 $f’(\xi) = -\frac{f(\xi)}{\xi}$。

证明步骤（可直接用于笔记/答题）：

  1. 构造辅助函数：将待证式变形为 $\xi f’(\xi) + f(\xi) = 0$，观察到左边是 $[x f(x)]’$ 的形式，故令 $F(x) = x f(x)$。
  2. 验证罗尔定理条件：
    • 连续性：$f(x)$ 连续、$x$ 连续，乘积 $F(x) \in C[0,1]$；
    • 可导性：乘积求导得 $F’(x) = f(x) + x f’(x)$，故 $F(x) \in D(0,1)$；
    • 端点相等：$F(0) = 0 \cdot f(0) = 0$，$F(1) = 1 \cdot f(1) = 0$，即 $F(0) = F(1)$。
  3. 应用定理：由罗尔定理，存在 $\xi \in (0,1)$ 使得 $F’(\xi) = 0$，即 $f(\xi) + \xi f’(\xi) = 0$，整理得 $f’(\xi) = -\frac{f(\xi)}{\xi}$。

二、拉格朗日中值定理（Lagrange’s Mean Value Theorem）

1. 核心定义

若函数 $f(x)$ 满足 2个条件：

  • 闭区间连续：$f(x) \in C[a,b]$；
  • 开区间可导：$f(x) \in D(a,b)$；
    则存在 $\xi \in (a,b)$，使得 $f’(\xi) = \frac{f(b) - f(a)}{b - a}$。

几何意义：连续可导曲线在区间上至少有一点的切线与区间端点连线（割线）平行。

常用等价形式：$f(b) - f(a) = f’(\xi)(b - a)$（增量公式，可直接用于不等式证明、函数单调性判断）。

2. 考研真题示例（2019年数学一）

设函数 $f(x)$ 在$[0,1]$上具有二阶导数，且 $f(1) = 0$。记 $g(x) = x f(x)$，证明：存在 $\xi \in (0,1)$，使得 $g’’(\xi) = 0$。

证明步骤（可直接用于笔记/答题）：

  1. 第一步：对 $g(x)$ 用罗尔定理找 $g’(x)$ 的零点：
    • $g(x) = x f(x)$，则 $g(0) = 0 \cdot f(0) = 0$，$g(1) = 1 \cdot f(1) = 0$；
    • 由罗尔定理，存在 $\eta \in (0,1)$ 使得 $g’(\eta) = 0$。
  2. 第二步：分析 $g’(x)$ 的表达式：
    • 求导得 $g’(x) = f(x) + x f’(x)$，因 $f(x)$ 二阶可导，故 $g’(x) \in C[0,\eta]$ 且 $g’(x) \in D(0,\eta)$（可导必连续）。
  3. 第三步：对 $g’(x)$ 用罗尔定理：
    • 代入 $x=0$ 得 $g’(0) = f(0) + 0 \cdot f’(0) = f(0)$，结合 $g’(\eta) = 0$，且 $g’(x)$ 可导，故存在 $\xi \in (0,\eta) \subset (0,1)$ 使得 $g’’(\xi) = 0$。

简化记忆：二阶导数为零的证明，常通过“对一阶导数用罗尔定理”，核心是找到一阶导数的两个零点。

三、柯西中值定理（Cauchy’s Mean Value Theorem）

1. 核心定义

若函数 $f(x)$ 和 $g(x)$ 满足 4个条件：

  • 闭区间连续：$f(x), g(x) \in C[a,b]$；
  • 开区间可导：$f(x), g(x) \in D(a,b)$；
  • 导数不同时为零：对任意 $x \in (a,b)$，$f’(x)^2 + g’(x)^2 \neq 0$；
  • 端点函数值不等：$g(a) \neq g(b)$；
    则存在 $\xi \in (a,b)$，使得 $\frac{f(b) - f(a)}{g(b) - g(a)} = \frac{f’(\xi)}{g’(\xi)}$。

几何意义：以 $x$ 为参数的曲线 $\begin{cases} X = g(x) \ Y = f(x) \end{cases}$ 上，至少有一点的切线与区间端点连线（割线）平行。

特殊情况：当 $g(x) = x$ 时，柯西中值定理退化为拉格朗日中值定理（故拉格朗日是柯西的特例）。

2. 考研真题示例（2018年数学二）

设函数 $f(x)$ 在$[a,b]$上连续，在$(a,b)$内可导，且 $f(a) = f(b) = 0$。证明：存在 $\xi \in (a,b)$，使得 $f’(\xi) = f(\xi) \cdot \frac{g’(\xi)}{g(\xi)}$（其中 $g(x)$ 在$[a,b]$上连续且不为零，$g’(x)$ 存在）。

证明步骤（可直接用于笔记/答题）：

  1. 构造辅助函数：将待证式变形为 $f’(\xi) g(\xi) - f(\xi) g’(\xi) = 0$，观察到左边是 $\left[ \frac{f(x)}{g(x)} \right]’$ 的分子（商的求导法则：$\left( \frac{f}{g} \right)’ = \frac{f’g - fg’}{g^2}$），故令 $F(x) = \frac{f(x)}{g(x)}$。
  2. 验证柯西中值定理条件：
    • 连续性：$f(x)$ 连续、$g(x)$ 连续且不为零，故 $F(x) \in C[a,b]$；
    • 可导性：商的求导法则，$F’(x) = \frac{f’(x) g(x) - f(x) g’(x)}{g^2(x)}$，故 $F(x) \in D(a,b)$；
    • 端点相等：$F(a) = \frac{f(a)}{g(a)} = 0$，$F(b) = \frac{f(b)}{g(b)} = 0$，即 $F(a) = F(b)$。
  3. 应用定理：由罗尔定理（或柯西中值定理，因 $g(x)$ 满足条件），存在 $\xi \in (a,b)$ 使得 $F’(\xi) = 0$，即 $f’(\xi) g(\xi) - f(\xi) g’(\xi) = 0$，整理得 $f’(\xi) = f(\xi) \cdot \frac{g’(\xi)}{g(\xi)}$。

关键技巧：柯西中值定理的核心是“构造商式辅助函数”，本质是解决“两个函数的增量比与导数比”的关系问题。

总结（便于笔记记忆）

定理 核心条件 核心结论 考研常用技巧
罗尔定理 连续、可导、端点值相等 $f’(\xi) = 0$ 构造辅助函数（乘积、加减）
拉格朗日定理 连续、可导 $f’(\xi) = \frac{f(b)-f(a)}{b-a}$ 增量公式、不等式证明
柯西定理 两函数连续、可导、$g(a) \neq g(b)$ $\frac{f(b)-f(a)}{g(b)-g(a)} = \frac{f’(\xi)}{g’(\xi)}$ 构造商式辅助函数

共性：三大中值定理均是“连续+可导”的核心条件，本质是建立函数增量与导数的联系，考研重点在于“辅助函数构造”和“定理嵌套使用”（如拉格朗日定理嵌套罗尔定理）。

泰勒公式和中值定理的联系

泰勒公式与三大中值定理（罗尔、拉格朗日、柯西）本质是 “局部近似”与“区间中值”的递进关系——泰勒公式是中值定理的“高阶推广”，中值定理是泰勒公式的“低阶特例”，核心联系如下：

1. 拉格朗日中值定理是泰勒公式的1阶特例

泰勒公式（带拉格朗日余项）的一般形式：
$f(x) = f(x_0) + f’(x_0)(x - x_0) + \frac{f’’(x_0)}{2!}(x - x_0)^2 + \dots + \frac{f^{(n)}(x_0)}{n!}(x - x_0)^n + R_n(x)$
其中拉格朗日余项 $R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!}(x - x_0)^{n+1}$（$\xi$ 在 $x_0$ 与 $x$ 之间）。

当取 $n=0$ 时（0阶泰勒公式）：
$f(x) = f(x_0) + f’(\xi)(x - x_0)$
令 $x = b$、$x_0 = a$，即得拉格朗日中值定理：$f(b) - f(a) = f’(\xi)(b - a)$。

2. 罗尔定理是泰勒公式的“特殊推论”

对1阶泰勒公式，若 $f(a) = f(b)$，则：
$f(b) = f(a) + f’(a)(b - a) + \frac{f’’(\xi)}{2!}(b - a)^2$
因 $f(a) = f(b)$，整理得 $f’(a) + \frac{f’’(\xi)}{2!}(b - a) = 0$——但更直接的逻辑是：罗尔定理是证明泰勒公式余项的“基础工具”（证明泰勒余项时，常构造辅助函数，对其应用罗尔定理推导余项形式）。

3. 柯西中值定理与泰勒公式的“互补关系”

  • 柯西中值定理聚焦 “两个函数的增量比与导数比”（解决比值型问题）；
  • 泰勒公式聚焦 “单个函数的高阶近似”（解决精度要求高的问题，如极限计算、不等式证明、高阶导数相关问题）。

但二者本质一致：均依赖“函数连续可导”的条件，均通过“中值 $\xi$”建立函数与导数的联系，且柯西中值定理可作为证明泰勒公式的辅助工具（如推导柯西型余项时会用到）。

4. 核心共性：“中值思想”的统一

无论是中值定理还是泰勒公式，核心都是 “在区间内存在一点 $\xi$，将函数的整体增量（或近似误差）用某阶导数表示”——中值定理是“1阶导数的整体关联”，泰勒公式是“多阶导数的局部近似”，前者是后者的简化，后者是前者的深化。
