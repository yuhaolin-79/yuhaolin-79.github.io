01: 伪回文数

描述

输入一个正整数，判断它是不是一个伪回文数。

  • 回文数：顺着看和倒着看都一样的数（例如 1221，12321）。
  • 伪回文数：改动一个数字就能变成回文数的数。
  • 注意：1位数不是伪回文数。

输入

第一行输入一个正整数 $nn$，表示下面有 $nn$ 组数据。
每组数据为一个正整数（能用 int 保存），占用一行。

输出

对于每组数据，若是伪回文数，输出 yes，否则输出 no。

描述

输入

03: 密钥格式化

描述

给定一个非空密钥字符串 S（包含字母、数字、破折号-），以及正整数 K，需按照以下规则重新格式化字符串：

  1. 仅保留 S 中的字母和数字，同时将所有小写字母转换为大写字母；
  2. 重新分组：第一个分组的字符数**≤K且≥1**，后续每个分组恰好包含 K 个字符；
  3. 分组之间用破折号-隔开。

输入

一个非空字符串 S 和一个正整数 K。

输出

按照规则格式化后的密钥字符串。

04: 今天星期几

描述

输入

1900年1月1日之后的合法日期，格式为yyyy-MM-dd（例如2000-01-01）。

输出

输入日期对应的星期（以英文单词形式输出，如Saturday）。

样例

输入 输出
2017-11-11 Saturday

描述

小H需要判断浏览器窗口是否能完全遮住原神窗口，窗口位置由4个整数定义：

输出

06: 统计学生信息

描述

定义学生信息结构体struct Student，包含以下字段：

  • name[20]：姓名
  • id：学号
  • math：数学分数
  • chinese：语文分数
  • sum：总分（需计算后赋值）

输入

第一行输入整数n（1 ≤ n ≤ 10），表示学生数量。
后续n行，每行输入一个学生的信息：姓名、学号、数学分数、语文分数。

输出

  1. 计算每个学生的总分（sum = math + chinese），并记录到sum字段；
  2. 按以下规则排序后输出每个学生的姓名和总分：
    • 优先按sum从大到小排序；
    • 若sum相同，则按id从小到大排序。

样例

输入 输出
3
Zhangsan 1 90 90
Lisi 2 90 87
Wanger 3 90 90		 Zhangsan 180
Wanger 180
Lisi 177
